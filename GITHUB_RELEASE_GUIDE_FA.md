# انتشار نسخه 0.6.0 در GitHub

1. محتویات پوشه RahaTVBrowser را در ریشه Repository جایگزین کنید.
2. Commit: `Release RahaTVBrowser 0.6.0`
3. در Actions اجرای `RahaTVBrowser build` را باز کنید.
4. Artifact با نام `raha-tv-browser-debug-and-unsigned-bundle` را دانلود کنید.
5. برای Release تگ `v0.6.0` بسازید و چهار خروجی APK/AAB و SHA256SUMS را پیوست کنید.
6. تا پایان تست تلویزیون واقعی، Release را Pre-release علامت بزنید.
