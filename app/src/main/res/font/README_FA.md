# افزودن وزیرمتن به RahaTVBrowser

به‌دلیل اینکه فایل باینری فونت داخل بسته توزیع نشده است، یکی از این دو حالت را اضافه کنید:

## حالت کم‌حجم پیشنهادی
فایل Variable فونت را با نام دقیق زیر قرار دهید:

`app/src/main/res/font/vazirmatn.ttf`

## حالت Regular/Bold
دو فایل را با این نام‌ها قرار دهید:

- `vazirmatn_regular.ttf`
- `vazirmatn_bold.ttf`

و فایل `vazirmatn.xml` را در همین پوشه بسازید. نام منبع نهایی باید `vazirmatn` باشد.

کد نسخه 0.4.1 فونت را در زمان اجرا پیدا می‌کند و روی رابط برنامه اعمال می‌کند. اگر فونت وجود نداشته باشد، برنامه بدون خطا از فونت پیش‌فرض Android استفاده می‌کند.
