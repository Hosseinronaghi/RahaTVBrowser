# ممیزی استاتیک RahaTVBrowser 0.6.0 R3

موارد بررسی‌شده:

- همه فایل‌های XML بدون خطای نحوی Parse شدند.
- منبع تکراری در پوشه‌های res پیدا نشد.
- فایل نامعتبر با پسوند tmp/bak/md/txt در res وجود ندارد.
- دکمه حالت خصوصی دیگر تب تازه اضافه نمی‌کند و همان تب را بین عادی/خصوصی بازسازی می‌کند.
- نوار ابزار به دو سطح تقسیم شده تا دکمه‌ها روی هم نیفتند.
- صفحه خانه RTL/LTR از DOM مشترک استفاده می‌کند و رویدادهای موس صریحاً فعال‌اند.
- حالت دسکتاپ و موبایل User-Agent، viewport، overview و initial scale را هماهنگ تغییر می‌دهد.
- پلیر دکمه بازگشت، بارگذاری زیرنویس خارجی، فونت وزیرمتن، اندازه و سبک زیرنویس، سرعت و نسبت تصویر دارد.
- IPTV نشانی‌های قبلی را محلی ذخیره می‌کند، جستجوی کانال دارد و پخش در همان صفحه کنار فهرست کانال‌ها انجام می‌شود.
- versionCode برابر 18 است.

محدودیت ممیزی:

Android SDK و Gradle در محیط بسته‌بندی در دسترس نبود؛ بنابراین کامپایل نهایی باید با GitHub Actions تأیید شود. ممیزی استاتیک جایگزین تست واقعی روی چند مدل Android TV نیست.
