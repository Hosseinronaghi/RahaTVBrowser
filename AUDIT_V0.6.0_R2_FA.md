# ممیزی اصلاحی RahaTVBrowser 0.6.0 R2

اصلاحات این بسته:

- بارگذاری وزیرمتن از `app/src/main/res/font/vazirmatn.ttf` در رابط بومی و صفحه خانه، در صورت وجود فایل فونت در مخزن.
- حذف مداخله نشانگر مجازی با رویدادهای موس و تبدیل میان‌برهای صفحه خانه به دکمه‌های Bridge-based.
- تقویت تفاوت حالت موبایل و دسکتاپ با User-Agent، viewport، zoom و بارگذاری مجدد واقعی.
- افزودن دکمه مستقل پوسته روشن/تیره؛ حالت خصوصی دیگر با آیکون پوسته اشتباه نمی‌شود.
- افزودن پیمایش D-pad در لبه‌های صفحه و Page/Channel Up/Down.
- افزودن مرورگر داخلی حافظه و USB به‌عنوان جایگزین SAF در تلویزیون‌های فاقد File Picker.
- افزودن دکمه بستن و پوشه بالاتر در Local Media.
- پشتیبانی IPTV از هدرهای M3U، `#EXTVLCOPT` و هدرهای پس از `|`، به‌همراه پیام خطای دقیق‌تر.
- تشخیص MIME بهتر برای HLS، DASH و MPEG-TS.
- جایگزینی آیکون و TV Banner.
- ادغام اصلاح رنگ پس‌زمینه و Workflow مبتنی بر Node.js 24.

بررسی‌های انجام‌شده:

- Parse تمام XMLها
- بررسی نام‌های تکراری منابع Android
- بررسی پسوندهای نامعتبر در `res`
- بررسی نبود `data:text/html` و `?attr/colorBackground`
- بررسی ساختار HTML صفحه خانه
- بررسی نحوی اولیه Java با `javac`؛ محیط حاضر Android SDK ندارد، بنابراین تأیید نهایی کامپایل با GitHub Actions انجام می‌شود.

نکته فونت: فایل باینری فونت در این بسته تعبیه نشده است. فایل موجود مخزن را با نام دقیق `vazirmatn.ttf` در مسیر `app/src/main/res/font/` نگه دارید.
