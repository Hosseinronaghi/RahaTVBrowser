# Raha Browser

این پروژه نسخه‌ی اولیه‌ی یک مرورگر متن‌باز مخصوص Android TV است که از موتور Mozilla GeckoView استفاده می‌کند.

## نام برنامه

نام رسمی برنامه **Raha Browser** است. موتور مرورگر همچنان Mozilla GeckoView است.


## شناسه برنامه

شناسه دائمی برنامه برای ساخت و انتشار:

```text
com.raha.browser.tv
```

پس از نخستین انتشار در Google Play این شناسه نباید تغییر کند.

## امکانات نسخه ۰٫۲٫۲

- اجرای صفحات با موتور مستقل Gecko/Firefox
- نمایش در لانچر Android TV
- نوار ابزار قابل کنترل با ریموت
- نشانگر مجازی با کلیدهای جهت‌دار و دکمه OK
- بازگشت، جلو، خانه و بارگذاری مجدد
- صفحهٔ خانه و جست‌وجوی پیش‌فرض Google
- مدیریت ورود و خروج ویدئو از حالت تمام‌صفحه
- دریافت لینک‌های HTTP و HTTPS از برنامه‌های دیگر
- متن‌های فارسی و انگلیسی
- پشتیبانی مستقیم از موس USB/بلوتوث و کلیک، Hover و اسکرول
- پشتیبانی از کیبورد USB/بلوتوث و تایپ در فرم‌های وب
- میان‌برهای Ctrl+L یا Ctrl+K، Ctrl+R، Alt+Left/Right و کلیدهای Browser

## ساخت APK در Android Studio

۱. پوشه پروژه را در Android Studio باز کنید.  
۲. JDK پروژه را روی 17 قرار دهید.  
۳. اجازه دهید Gradle وابستگی GeckoView را دریافت کند.  
۴. از مسیر `Build > Build APK(s)` خروجی بگیرید.

مسیر خروجی:

```text
app/build/outputs/apk/debug/app-debug.apk
```

برای گوگل‌پلی باید AAB بسازید:

```text
Build > Generate Signed Bundle / APK > Android App Bundle
```

## ساخت خودکار بدون Android Studio

راهنمای `BUILD_APK_GITHUB_FA.md` را ببینید. Workflow موجود، APK آزمایشی امضاشده و AAB بدون امضای انتشار تولید می‌کند.

## هشدار

این نسخه یک MVP قابل توسعه است، نه مرورگر نهایی آماده‌ی انتشار عمومی. پیش از انتشار باید دانلود، مجوزهای سایت، هشدارهای امنیتی، تاریخچه، نشانک‌ها، تست‌های دستگاه و سیاست حریم خصوصی تکمیل شوند.

وضعیت دقیق اعتبارسنجی در `BUILD_STATUS_FA.md` ثبت شده است.
