# تغییرات RahaTVBrowser 0.4.0

- دو APK مستقل برای `arm64-v8a` و `armeabi-v7a`.
- پخش‌کننده داخلی Media3/ExoPlayer برای لینک‌های مستقیم M3U8، MPD، MP4، WebM و MKV.
- استفاده از SurfaceView در تمام‌صفحه برای کاهش فشار رندر و بهبود شانس پخش روان 1080p/4K.
- حداکثر ۵ تب و بازکردن `target=_blank` در تب جدید.
- تب خصوصی با GeckoSession خصوصی و بدون ثبت History.
- تنظیمات اولیه: Desktop/Mobile، سرعت نشانگر، پاک‌کردن History و بستن تب‌ها.
- ذخیره و نمایش ۱۰ بازدید آخر.
- صفحه خانه متراکم‌تر با ۸ علاقه‌مندی در هر ردیف Full HD.
- دکمه بستن تب، پخش‌کننده، تنظیمات، تب‌ها و حالت خصوصی.
- دکمه‌های کوچک‌تر و نوار ابزار کم‌ارتفاع‌تر.
- Desktop Mode واقعی با تغییر هم‌زمان User Agent و Viewport و Reload.
- قابلیت امضای ثابت از طریق GitHub Secrets برای Update نسخه‌های آینده.

## محدودیت‌ها

- پخش‌کننده داخلی فقط لینک مستقیم رسانه را می‌پذیرد. JW Player جاسازی‌شده داخل صفحه همچنان در GeckoView اجرا می‌شود.
- 4K روان به Codec، Decoder سخت‌افزاری، پهنای باند و توان دستگاه وابسته است.
- کلیک موس روی صفحه‌کلید سیستمی به پیاده‌سازی IME سازنده تلویزیون وابسته است؛ نوار آدرس خود برنامه با موس قابل استفاده است.
- فونت Vazirmatn در بسته قرار نگرفته است؛ برای افزودن آن باید فایل فونت دارای مجوز را خودتان در پروژه قرار دهید.
